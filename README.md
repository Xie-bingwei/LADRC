# LADRC

采用的是线性化的三阶ESO，控制律采用的是PD，通过TD提高跟踪速度与精度

转成C/C++就可直接在嵌入式系统上运行，LADRC在智能车，飞控，机器人连杆控制，ROV的水下控制等情况都有非常不错的稳定性，调参我自己感觉比PID要简单一些(前提是要知道原理)，如果不懂原理，也能试出比较好的效果，但是可能要多花点时间

之后会抽个时间在Readme上写一写该控制器的原理推导和调参方法
